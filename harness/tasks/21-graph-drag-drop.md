# Task 21: 그래프 드래그 앤 드롭

## 배경

현재 그래프는 현재 네트워크 뷰에 포함된 친구들만 노드로 표시된다.
circleSize 뷰 밖에 있는 친구를 그래프에 직접 추가해 네트워크 탐색을 할 수 있어야 한다.

## UX 흐름

```
그래프 사이드 패널에 "친구 목록" 섹션 추가
  → 현재 뷰에 없는 친구들 목록 표시
  → 친구 카드를 그래프 캔버스로 드래그 앤 드롭
  → 해당 친구 노드가 그래프에 동적 추가
  → 노드 클릭 → getOneHopMutualFriendEdges()로 현재 네트워크와의 엣지 로드
```

## 기술 고려사항

- `friendsList` 전체에서 현재 그래프에 없는 친구 필터링 → 드롭 대상 목록
- 추가된 노드: Task 08 suggestion 노드처럼 별도 시각 스타일 (임시 추가 표시)
- 빈 영역 클릭 시 임시 노드 제거 여부 — PLAN 단계에서 결정
- Cytoscape 드래그 앤 드롭 구현 방식 — PLAN 단계에서 결정

## 스코프 외

- 드롭한 노드의 영구 저장 (circleSize 변경)

## 검증

- 사이드 패널에 현재 뷰 밖 친구 목록 표시
- 드래그 앤 드롭 → 노드 그래프에 추가
- 추가된 노드 클릭 → 현재 네트워크와의 엣지 렌더링
- `npx tsc --noEmit` 에러 없음
